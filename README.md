# Springboot_Vue_Supermarket_Sales_Management
基于Springboot+Vue超市销售管理系统设计毕业源码案例设计

开发软件：Eclipse/Idea + WebStorm/VsCode
数据库：mysql

  今天带来的是一个基于Springboot+Vue的前后端分离的超市销售管理系统，系统一共有系统管理员，会计，销售员，库管员几个角色。管理员登录后可以拥有最高权限看，包括管理添加员工信息，选择商品和会员自动结算商品，其中销售的时候还可以参与积分打折活动，查询销售订单历史记录，处理商品退货，统计商品销售信息和统计商品信息，添加管理会员信息，添加维护商品信息。会计登录后主要是财务统计功能，销售员登录后主要是销售产品，库管员登录后主要是管理商品库存。
